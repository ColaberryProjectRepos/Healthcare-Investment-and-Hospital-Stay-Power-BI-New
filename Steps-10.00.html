Group the Location Column into Regions 10


  • Go to Home page
  • Select the Location column 
  • Then select Column tools on the home ribbon
  • Click on Data Groups then new data groups
  • Name the new column Region
  • Group the countries according to the screen shots below by following the GIF
  • When done grouping all the regions and Locations click okay and a new column named region will be created
